7 Hobbies Overview

Unlock order · Mentors · Mechanics · Tips

1️⃣ Tutorial

Fishing + Gardening (auto)

2️⃣ DG Lv6

Cooking first (best profit)

3️⃣ DG Lv6+

Bugs + Birds (ticket)

4️⃣ DG Lv12

Cat Care → Dog Care

🎫 Each hobby needs a Hobby Expansion Ticket (tutorial quests / DG guild rank / story — scarce, choose wisely).

钓鱼

Fishing

🧑‍🏫Mentor：凡妮亚（Vanya）
🔓教学自动解锁
📊约 94 种鱼

在湖泊、河流、海洋钓捕各种鱼类，掌握线张力，追猎稀有的彩虹鱼。

  • • 注意线张力，太紧会断线
  • • 雨天、彩虹天出没稀有鱼
  • • 不同地点/时段/天气对应不同鱼种

园艺

Gardening

🧑‍🏫Mentor：布朗克（Blanc）
🔓教学自动解锁
📊13+ 作物 · 含花杂交

种植各类作物（番茄到牛油果），浇水、收获、利润最大化；并包含花杂交系统。

  • • 每日浇水保持生长
  • • 花杂交可培育 1★→5★ 稀有花
  • • 高星花售价与园艺经验更高
View Guide →
🍳

烹饪

Cooking

🧑‍🏫Mentor：马西莫（Massimo）
🔓DG 等级 6（推荐首选）
📊约 98 道食谱

最赚钱的爱好！把钓鱼、园艺的原料做成菜肴，利润可达 2-3 倍金币。

  • • 优先解锁，与钓鱼/园艺协同
  • • 用自产原料做菜利润最高
  • • 关注食谱解锁等级
View Guide →

捕虫

Insect Catching

🧑‍🏫Mentor：难波（Naniwa）
🔓DG 等级 6+（需爱好扩展券）
📊约 76 种昆虫

用泡泡网捕捉蝴蝶、飞蛾、甲虫等昆虫，收藏爱好者的最爱。

  • • 不同虫有特定时段与天气
  • • 靠近时动作要轻
  • • 部分昆虫稀有度高

观鸟

Birdwatching

🧑‍🏫Mentor：贝莉（Bailey J）
🔓DG 等级 6+（需爱好扩展券）
📊约 68 种鸟

用扫描仪拍摄各种鸟类，回收信息卡换取奖励。

  • • 对焦准确拿高星级
  • • 信息卡可回收换奖励
  • • 注意鸟类出现时段

养猫

Cat Care

🧑‍🏫Mentor：琼太太（Mrs. Joan）
🔓DG 等级 12
📊约 14 个品种

收养并照顾猫咪：喂食、抚摸、洗澡、玩耍，培养感情。

  • • 每日喂食与互动提升好感
  • • 不同品种有不同性格
  • • 解锁后才能解锁养狗

养狗

Dog Care

🧑‍🏫Mentor：琼太太（Mrs. Joan）
🔓DG 等级 12（需先解锁养猫）
📊约 11 个品种

训练狗狗执行指令（如寻宝），拥有一位忠诚的伙伴。

  • • 必须先解锁养猫
  • • 训练寻宝指令可发现宝藏
  • • 每日遛狗互动
