Heartopia 7-Day Beginner Route: Day-by-Day

How to spend your first 7 days efficiently.

📌 Daily reset at server 6:00 AM. Verify in-game.

3 Core Principles

  1. Daily Requests first: Resident Requests = main Contribution Medal source (DG leveling), time-limited. ~10 medals + 5 Wishing Stars + 1000G each; ~50 medals + 5000G/day.
  2. Rush DG6 → Cooking: most profitable hobby.
  3. Don’t sell raw: process first (Money Route).

Daily Priorities

  1. 🔴 Clear Resident Requests (medals, limited)
  2. 🎁 Claim daily / codes
  3. 🌱 Water + harvest
  4. 🎣 Fish (weather: rain/rainbow priority)
  5. 🍄 Forage (free fruit/mushroom)
  6. 🍳 Cook (multiple stoves)
  7. 💰 Sell to Albert Jr.

Use Daily Checklist.

Day-by-Day

Day 1: Tutorial + Start

  • Follow tutorial, auto-unlock Fishing + Gardening.
  • Free fruit → jam → sell.
  • Daily requests.
  • Goal: learn basics, first gold.

Day 2-3: Save + Stock

  • Clear requests (DG medals).
  • Plant Wheat + Potato.
  • Fish (rainy Carp) → cook.
  • Goal: near DG6, stock materials.

Day 4-5: DG6 + Cooking

  • Clear requests + cook for gold.
  • Hit DG6 → unlock Cooking (first choice).
  • Find Massimo, learn cooking, buy recipes.
  • Multiple stoves.
  • Goal: Cooking unlocked, steady income.

Day 6-7: Scale + Hobbies

  • Level cooking, high-profit dishes (Latte, Grape Jam — use Recipe Calc).
  • Unlock Bug Catching / Birdwatching (DG6+).
  • Stock Grapes/Strawberries.
  • Goal: steady daily income.

Milestones

StageGoal
DG 1-5Fishing + Gardening start
DG 6Unlock Cooking
DG 6+Bugs / Birds
DG 12Cat → Dog

Energy

  • 😴 Sleep (slow, free, AFK).
  • 🍲 Eat low-quality dishes (sell the good ones).
  • Fishing/chopping/mining costs energy — plan.

One-liner

Daily requests → DG6 Cooking → don’t sell raw → cook for gold.
