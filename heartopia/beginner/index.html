🎮 Core Gameplay

Heartopia is a life sim centered on 7 hobbies + building + social:

  • 🎣 Fishing (auto-unlock) — ~97 fish
  • 🌱 Gardening (auto-unlock) — crops + flower crossbreeding (1★→5★)
  • 🍳 Cooking (DG6, most profitable) — ~77-98 recipes
  • 🦋 Bug Catching / 📷 Birdwatching (DG6+)
  • 🐱 Cat Care / 🐶 Dog Care (DG12, cat first)
  • 🏠 Build / decorate home, 🤝 befriend residents

Unlock order & routes: 7 Hobbies Overview.

🚀 First Few Days

Full day-by-day route: 7-Day Route. Key principles:

  1. Tutorial: auto-unlocks Fishing + Gardening.
  2. Daily Requests first: main source of Contribution Medals (DG leveling), time-limited.
  3. Rush DG6 to unlock Cooking (most profitable).
  4. Don’t sell raw materials: process into dishes for 2-3× profit (Money Route).

💰 Making Gold

See Money Route. Core: Cooking + Fishing + Gardening trinity, sell processed (golden rule: never sell raw).

🎯 Beginner Checklist

  • Claim daily / Codes
  • Finish tutorial (Fishing + Gardening)
  • Clear Daily Requests daily (medals)
  • Reach DG6 → unlock Cooking
  • Process materials before selling
  • Unlock bugs / birds / pets as tickets allow

❓ FAQ

Not enough gold? Unlock Cooking (DG6), process materials — see Money Route.

How to level DG? Daily Requests = Contribution Medals (main source, time-limited).

Can’t catch rare fish? Time + weather (rain / rainbow = higher rare rate), upgrade rod — see Fishing Guide.
