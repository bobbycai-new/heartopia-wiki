Heartopia Bug Catching: Stealth, Net Upgrades & All Insects

Bug catching is a collection hobby — catch butterflies, beetles, dragonflies to fill your journal.

📌 Mechanics from community research. Insect count varies (~48-76). Verify in-game.

1. Unlock

  • DG Level 6, use Hobby Expansion Ticket on Bug Catching.
  • Find mentor Naniwa (forest/garden), get Basic Bug Net.
  • Naniwa sells net upgrades, inflatables, repair kits.

2. Mechanics (Stealth + Timing)

  1. Equip net.
  2. Spot bugs on trees/flowers/grass/water.
  3. Walk slowly, don’t run — bugs detect & flee.
  4. Stop when net-range indicator appears.
  5. Swing. Success = net level + distance + rarity.

Bug fled? Leave area, return after spawn cooldown.

3. Net Levels

LevelUnlockEffect
BasicLv1Standard range, common bugs
IntermediateLv3+Larger range, better rare rate
MasterLv5+Max range, ultra-rare

4. Areas & Bugs

  • Central/Suburbs: Orange Tip, Brimstone, Blue Butterfly, Grasshopper, Green Birdwing (evening)
  • Flower Field: Asparagus Beetle, Bumblebee, Katydid, Purple Emperor, Peacock
  • Forest: Swallowtail, Wasp Beetle, Cicada, Ladybug, Stag Beetle, Demoiselle (night/rain)
  • Fishing Village: Cabbage White, Grasshopper, Ant, Leopard, Shield Bug
  • Onsen Mountain: alpine species

❄️ Winter: seasonal “Frostspore” butterflies.

5. Time & Weather

  • Night bugs: beetles, longicorns.
  • Rain/Rainbow: Demoiselle, Ladybug more common.

6. Tips

  1. Don’t run — sneak.
  2. Upgrade net.
  3. Plan route by area + time + weather.
  4. Bug fled → leave & return.
  5. Don’t miss winter seasonal.
