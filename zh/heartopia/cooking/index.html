心动小镇烹饪攻略：升级、食谱与最赚钱菜品

烹饪是心动小镇最赚钱的爱好——把钓鱼、园艺的原料做成菜，利润能翻 2-3 倍。本篇覆盖解锁、机制、食材、升级路线与高利润食谱。

📌 数据声明：机制与利润基于社区整理。食谱总数各源不一（约 77–98 道），已标注；具体售价以游戏内为准，持续校验。

一、解锁条件

  • 小镇发展等级达到 DG Level 6。
  • 到镇中心餐厅找 马西莫（Massimo），完成他的烹饪教学任务。
  • 使用餐厅的公共炉灶，或自己造一个。

💡 推荐：DG6 可在烹饪 / 捕虫 / 观鸟中三选一，首选烹饪——它与钓鱼、园艺完美协同，把原料变高利润菜品，是早期最佳收入来源。

二、烹饪机制

1. 操作流程

与炉灶互动 → 选已学食谱 → 放入食材 → QTE 小游戏控火 → 收菜。

2. ⚠️ 小心火候

QTE 失败会把菜烧成「奇怪的食物 / 饮料」，售价大打折扣——专注控火，别走神。

3. 星级

菜品有 1–5 星，星级越高售价越高，尽量冲高星。

4. 多炉灶批量

在工作台用木材 + 矿石打造额外炉灶，多灶同煮，量产提速。

三、食材来源

来源食材备注
🛒 找马西莫买（不可自产）牛奶 ~50 / 蛋 ~100 / 奶酪 ~100 / 黄油 ~150部分食谱必需
🌱 园艺自种小麦、番茄、土豆、玉米、草莓、胡萝卜、生菜、咖啡豆主力食材
🎣 钓鱼获得任意鱼、贝壳类、鲑鱼、金枪鱼、龙虾海鲜食谱
🍄 野外采集蘑菇（多种）、野果免费
🍎 免费水果苹果、覆盆子、蓝莓、橘子做果酱首选

四、升级路线

烹饪等级决定可做食谱，做菜攒经验升级。各等级大致所需经验：

等级累计经验解锁方向
入门~100基础（面包、煎蛋）
2–3~250–500沙拉、简单汤
4–5~850–1300烧烤、意面、寿司
6~1900拿铁、葡萄果酱（高利润）
7–8~2700–3700提拉米苏、高级菜
10+5000+大师食谱、收藏认证

⚡ 升级技巧：多灶同煮；早期多做耗时短的食谱（单位时间经验更高）。

五、高利润食谱（重点）

以下为社区整理的高利润菜品，售价以游戏内为准。

菜品亮点
🍓 草莓酱（Strawberry Jam）被广泛认为利润最高，单份可达约 12,640 金币
🍇 葡萄果酱（Grape Jam）Lv6 解锁，高利润
☕ 拿铁（Latte）Lv6 解锁，稳定收益
🍄 烤蘑菇（Grilled Mushroom）早期盈利选择
🍰 提拉米苏（Tiramisu）高级甜点，高售价

起步食谱（自动学会）

  • 🥗 家常沙拉 = 任意蔬菜 ×2
  • 🍯 综合果酱 = 混合水果 ×4

Lv1 食谱（找马西莫购买）

  • 🍟 炸鱼薯条 = 任意鱼 ×2 + 土豆 ×2
  • 🍰 芝士蛋糕 = 奶酪 + 牛奶 + 小麦
  • 🍄 烤蘑菇 = 任意蘑菇 ×4

六、赚钱技巧总结

  1. 首选烹饪：与钓鱼、园艺协同，原料自给。
  2. 多灶量产：造多个炉灶同时开煮。
  3. 专注高利润菜：草莓酱、葡萄果酱、拿铁。
  4. 别烧焦：QTE 控火，烧焦贬值。
  5. 冲高星：高星菜品售价翻倍。
  6. 免费水果做果酱：苹果 / 莓果做果酱，零成本高利润。
