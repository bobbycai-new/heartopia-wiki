潜行、网升级与全昆虫捕捉

心动小镇捕虫攻略：潜行、网升级与全昆虫

捕虫是收藏向爱好，用虫网捕捉蝴蝶、甲虫、蜻蜓等昆虫，填满你的昆虫图鉴。

📌 数据声明：机制基于社区整理；昆虫总数各源不一（约 48–76 种），以游戏内为准，持续校验。

一、解锁条件

  • 小镇发展等级达 DG Level 6，用爱好扩展券选「捕虫」。
  • 找导师 难波（Naniwa）（森林 / 花园区），他给你基础虫网并教学。
  • 难波还出售虫网升级、充气昆虫诱捕器、修理包。

二、捕捉机制（潜行 + 时机）

  1. 装备虫网。
  2. 在树、花、草、水边寻找虫的动静。
  3. 慢慢走过去，千万别跑——虫有侦测半径，跑动会吓跑它。
  4. 网范围指示器出现时停下。
  5. 挥网。成功率取决于网等级、距离、虫的稀有度。

虫跑了？离开区域一会儿再回来，刷新点冷却后会重置。

三、虫网等级

等级解锁效果
基础 BasicLv1标准范围，适合常见虫
中级 IntermediateLv3+更大范围，提升稀有虫成功率
大师 MasterLv5+最大范围，专抓超稀有

四、区域与虫种

各区有独特昆虫，随等级从镇区扩展到森林、山区：

  • 中心 / 近郊：橙尖蝶、黄粉蝶、蓝蝶、蚱蜢、绿鸟翼蝶（傍晚）
  • 花田：芦笋甲虫、熊蜂、螽斯、紫闪蝶、孔雀蛱蝶
  • 森林：金凤蝶、天牛甲虫、蝉、瓢虫、锹形虫、色蟌（夜行 / 雨天）
  • 渔村：菜粉蝶、蚂蚱、蚁、豹蝶、蓝盾蝽（码头夜 / 灯塔晨）
  • 温泉山：更多高山品种

❄️ 冬季有专属「霜孢」季节性蝴蝶，别错过。

五、时段与天气

  • 夜行虫：甲虫、天牛多在夜晚出没。
  • 雨天 / 彩虹天：色蟌、瓢虫等更易出现。
  • 每只虫都有特定的时段与天气条件，注意查图鉴。

六、技巧总结

  1. 别跑，潜行接近。
  2. 升级虫网，提升范围与稀有虫成功率。
  3. 按「区域 + 时段 + 天气」规划捕捉路线。
  4. 虫跑了就离开再回，等刷新。
  5. 冬季留意季节限定蝴蝶。
