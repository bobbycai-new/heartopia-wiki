心动小镇渔点与钓鱼攻略

钓鱼是教学自动解锁的爱好。本篇给渔点（栖息地）总览 + 钓鱼机制 + 技巧。具体每种鱼的位置 / 时间 / 天气 / 价格，请查 鱼类图鉴（可搜索筛选 + 收集追踪）。

一、渔点（栖息地）总览

游戏地图分为多个水域，不同水域出没不同鱼类：

水域类型特点
巨木江 / 晚霞江 / 宁静之河 / 浅滩河河流常见淡水鱼
近郊湖 / 森林湖 / 草原湖 / 温泉山湖湖泊淡水鱼，部分稀有
鲸之海 / 东海 / 九海 / 平静之海 / 海上垂钓点海域海鱼、贝壳类
任意河流 / 任意海域 / 任意湖泊通用水域多种鱼均可出没

完整「鱼 ↔ 地点」对应关系见 鱼类图鉴，可按「地点」筛选查看某水域的所有鱼。

二、钓鱼机制

1. 线张力

拉杆时注意线张力——太紧会断线。掌握节奏，张力接近极限时松一下再收。

2. 时段

不同鱼在不同时段出没（早 / 午 / 晚 / 夜）。夜行鱼多在夜晚，清晨 / 傍晚鱼类活跃。

3. 天气（关键）

  • 晴天：常见鱼活跃。
  • 雨天：稀有鱼出现率提升（如近郊湖雨天易钓鲫鱼）。
  • 彩虹天：稀有 / 特殊鱼出现，彩虹鱼在海域彩虹天可钓，售价极高。

三、稀有 / 高价鱼路线

  • 雨天 → 近郊湖钓鲫鱼（图鉴 fish-006，雨天出没），带回家做菜，价值翻倍（见 赚钱路线）。
  • 彩虹天 → 去海域（如鲸之海 / 平静之海）钓彩虹鱼，单条售价极高（稀有传说鱼，图鉴待补全）。
  • 传说级鱼（金色稀有）：售价最高，难钓，需要高等级鱼竿 + 特定条件。

四、技巧总结

  1. 注意线张力，别断线。
  2. 雨天 / 彩虹天优先出门钓鱼。
  3. 按「地点 + 时段 + 天气」找特定鱼（用 鱼类图鉴 筛选）。
  4. 升级鱼竿提升成功率与稀有鱼概率。
  5. 别直接卖生鱼——做菜利润翻倍。
