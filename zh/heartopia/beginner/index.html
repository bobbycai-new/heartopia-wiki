🎮 游戏核心

心动小镇是一款生活模拟游戏，核心玩法围绕 7 大爱好 + 建房 + 社交：

  • 🎣 钓鱼（教学自动解锁）—— 约 97 种鱼
  • 🌱 园艺（教学自动解锁）—— 种植作物 + 花杂交（1★→5★）
  • 🍳 烹饪（DG6 解锁，最赚钱）—— 约 77-98 道食谱
  • 🦋 捕虫 / 📷 观鸟（DG6+ 解锁）
  • 🐱 养猫 / 🐶 养狗（DG12 解锁，先猫后狗）
  • 🏠 建造 / 装饰家园、🤝 与居民交友

7 大爱好的解锁顺序 + 推荐路线见 七大爱好总览。

🚀 新手头几天怎么过

完整逐日路线见 新手 7 天路线。简要原则：

  1. 教学：跟引导走，自动解锁钓鱼 + 园艺。
  2. 每日请求优先：居民请求（Resident Requests）是贡献勋章（DG 升级）的主要来源，限时，每天先清。
  3. 冲 DG6 解锁烹饪：烹饪是最赚钱的爱好，第一时间解锁。
  4. 别卖原料：生鱼 / 生果 / 生菜加工成菜再卖，利润翻倍（见 赚钱路线）。

💰 怎么赚钱

详见 赚钱路线。核心一句话：烹饪 + 钓鱼 + 园艺三位一体，加工后出售（黄金法则：永远别卖原料）。

  • 鱼类真实售价查 鱼类图鉴
  • 食谱真实利润查 食谱利润计算器
  • 每日赚钱闭环用 每日清单 勾选

🎣 钓鱼

教学自动解锁。机制（线张力 / 时段 / 天气）+ 渔点见 渔点攻略；全部鱼类的真实数据（价格 / 地点 / 时间 / 天气）查 鱼类图鉴，还可勾选追踪收集进度。

🌱 园艺 / 花杂交

教学自动解锁。种作物 + 杂交花。花的 1★→5★ 配方与机制见 花杂交攻略。

🍳 烹饪

DG6 解锁（推荐首选爱好）。食谱、升级路线、高利润菜见 烹饪攻略；全食谱按利润排序查 食谱计算器。

🏠 建房 · 🐾 宠物

  • 建造 / 装饰：游戏核心自由度，规划好房间用途、预留扩展空间即可。
  • 宠物：猫 + 狗两种（DG12 解锁，必须先猫后狗），领养 / 性格 / 训练见 宠物攻略。

🎯 新手必做清单

  • 领每日奖励 / 查 兑换码
  • 完成教学（钓鱼 + 园艺自动解锁）
  • 每日清居民请求（贡献勋章）
  • 冲 DG6 解锁烹饪
  • 加工原料再卖（别卖生的）
  • 视情况解锁捕虫 / 观鸟 / 宠物

❓ 常见问题

金币不够？ 优先解锁烹饪（DG6），加工原料出售——见 赚钱路线。

怎么升级 DG？ 每日居民请求是贡献勋章的主要来源，限时必做。

钓鱼总钓不到稀有鱼？ 注意时段 + 天气（雨天 / 彩虹天稀有率高），升级鱼竿——见 渔点攻略。
