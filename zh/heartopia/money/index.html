心动小镇赚钱攻略：最快金币路线

想买新房图纸、扩仓库？本篇给完整的赚钱路线：黄金法则、财富三位一体（烹饪 / 园艺 / 钓鱼）、早中后期策略与每日闭环。

📌 数据声明：基于社区整理 + 本站食谱/鱼类数据。具体数值以游戏内为准，持续校验。

一、黄金法则：永远别卖原料

生鱼、生果、生菜直接卖 = 亏。 把它们加工成菜再卖，利润翻倍甚至更高。

一个生苹果直接卖只值几金币；做成苹果派 / 果酱能卖上百。加工是赚钱的核心。

二、在哪卖？找小艾伯特

心动小镇没有”出货箱”。要把东西换成金币，找金币商人 小艾伯特（Albert Jr.）：

  • 他在镇上四处走动（无固定店铺），打开地图追踪他的图标。
  • 交互选「出售物品」，可批量排队卖。
  • 钓鱼 / 做菜后，攒一批找他一次清空，省时间。

三、财富三位一体

🍳 烹饪（最快最稳）

  • 早期零成本：摇树 / 灌木收免费水果（苹果、橘子、莓果）→ 做果酱，纯利润。
  • 低星作物也拿来做菜（烹饪 minigame 有概率升星）。
  • 用本站 食谱利润计算器 查 Top 利润菜，按利润排序做最赚的。

🌱 园艺（原料引擎）

农场是烹饪的原料引擎，不是直接卖作物的地方：

  • 早期：种 小麦 + 土豆（中后期高利润食谱的必需原料）。
  • 后期：转 葡萄 + 草莓（果酱 / 甜点高利润）。
  • 时间管理：在线时种快作物；睡前种慢作物（葡萄 >10 小时）。
  • 黑松露：去森林岛采集（约 15 分钟刷新），是传奇料理的关键食材。

🎣 钓鱼（暴利）

  • 雨天：去近郊湖钓 鲫鱼（图鉴 fish-006，雨天出没），回家做菜，价值翻倍。
  • 彩虹天：去海域（如鲸之海 / 平静之海）钓 彩虹鱼，超高售价（稀有传说鱼，图鉴待补全）。
  • 别直接卖生鱼——做成菜再卖！

四、阶段路线

🌅 早期（DG 1–6）

  1. 摇树收免费水果 → 做果酱卖（零成本起步）。
  2. 钓鱼（雨天鲫鱼）→ 做菜。
  3. 种小麦 + 土豆囤原料。
  4. DG6 第一时间解锁烹饪，立刻开始做菜赚钱。

☀️ 中期（DG 6–12）

  1. 烹饪升级，解锁高利润食谱（拿铁、葡萄果酱等）。
  2. 多炉灶批量做菜，量产提速。
  3. 钓鱼冲稀有鱼种。
  4. 囤葡萄 / 草莓原料。

🌙 后期（DG 12+）

  1. 大师食谱（提拉米苏、至尊蛋糕等）高售价。
  2. 彩虹鱼 + 传奇料理。
  3. 黑松露系列料理。

五、每日赚钱闭环

  1. 领每日奖励 / 兑换码
  2. 浇水 + 收作物
  3. 钓鱼（看天气：雨天 / 彩虹优先）
  4. 采集（免费水果 / 蘑菇 / 松露）
  5. 批量做菜（多炉灶同时开）
  6. 找小艾伯特一次卖光

用本站 每日清单 勾选追踪，养成闭环习惯。

六、一句话结论

  • 首选烹饪，与钓鱼 / 园艺协同。
  • 别卖原料，全部加工。
  • 免费水果做果酱，零成本起步。
  • 雨天 / 彩虹钓鱼，暴利时刻。
  • 找小艾伯特批量卖，别一次次跑。
