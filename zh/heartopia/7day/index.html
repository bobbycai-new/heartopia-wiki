心动小镇新手 7 天路线：逐日决策清单

头 7 天怎么过最高效？本篇给逐日路线 + 每日优先级，帮你快速度过新手期、稳定赚钱。

📌 数据声明：基于社区整理。每日重置为服务器时间 6:00 AM；具体以游戏内为准。

三个核心原则

  1. 每日请求优先：居民请求（Resident Requests）是贡献勋章（DG 升级）的主要来源，限时，每天先清。每个请求约奖励 10 贡献勋章 + 5 许愿星 + 1000 金币，一天下来约 50 勋章 + 5000 金币。
  2. 冲 DG6 解锁烹饪：烹饪是最赚钱的爱好，DG6 一到立刻解锁。
  3. 别卖原料：生鱼 / 生果 / 生菜加工成菜再卖（详见 赚钱路线）。

每日优先级（每天都做）

  1. 🔴 清居民请求（贡献勋章，限时）
  2. 🎁 领每日奖励 / 兑换码
  3. 🌱 浇水 + 收作物
  4. 🎣 钓鱼（看天气：雨天 / 彩虹优先）
  5. 🍄 采集（免费水果 / 蘑菇）
  6. 🍳 做菜（多炉灶批量）
  7. 💰 找小艾伯特批量卖

用本站 每日清单 勾选追踪，养成闭环习惯。

逐日路线

Day 1：教学 + 起步

  • 跟教学走，自动解锁钓鱼 + 园艺。
  • 摇树收免费水果 → 做果酱卖（零成本起步金）。
  • 完成当日居民请求。
  • 🎯 目标：熟悉操作，攒第一笔金。

Day 2–3：攒钱 + 囤原料

  • 每日请求全清（贡献勋章冲 DG 等级）。
  • 种 小麦 + 土豆（中后期高利润食谱的必需原料）。
  • 钓鱼（雨天鲫鱼）→ 做菜卖。
  • 采集免费水果做果酱。
  • 🎯 目标：DG 等级接近 6，囤一波原料。

Day 4–5：冲 DG6 + 解锁烹饪

  • 持续清请求 + 做菜赚钱。
  • 达 DG6 → 立刻用爱好扩展券解锁烹饪（首选爱好）。
  • 找马西莫学烹饪，买基础食谱。
  • 开始多炉灶做菜。
  • 🎯 目标：解锁烹饪，开始稳定赚钱。

Day 6–7：扩大生产 + 解锁爱好

  • 烹饪升级，做高利润菜（拿铁、葡萄果酱等，用 食谱计算器 查）。
  • 视爱好扩展券情况解锁捕虫 / 观鸟（DG6+）。
  • 囤葡萄 / 草莓原料。
  • 升级工具 / 扩仓库。
  • 🎯 目标：稳定日入，为中期打基础。

关键里程碑

阶段目标
DG 1–5钓鱼 + 园艺起步，攒钱囤原料
DG 6解锁烹饪（首选爱好）
DG 6+解锁捕虫 / 观鸟（按扩展券）
DG 12解锁养猫 → 养狗

体力管理

  • 😴 睡觉恢复（慢，免费，适合挂机）。
  • 🍲 吃低质菜快速恢复（高质菜留着卖钱）。
  • 钓鱼 / 砍树 / 挖矿都耗体力，规划好顺序。

一句话总结

头 7 天：每日清请求 → 冲 DG6 解锁烹饪 → 别卖原料 → 做菜赚钱。
